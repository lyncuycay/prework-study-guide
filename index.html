HTML

  • HTML, Hypertext Markup Language is used to create the structure of a webpage
  • It is made up of element that provide content and basic framework of all the pages of the web
  • Types of elements are: Text, Image, Header and List
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • CSS, Cascading Style Sheets is a programming language used by developers to define how webpage content is presented to users
  • CSS defines the colors, aesthetics, visual layout and organization.
  • We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options

Git

  • Git is a version control system that let's you manage and keep track of each version of your project.
  • Git commands always start with git and are used to save and track versions of you code.
  • git status: checks what branch we are currentl on
  • git checkout -b branch name: creates a new branch and switches to it.

JavaScript

  • JavaScript is a programming language that allows users to interact with websites.
  • Its core features is Variables, which store data types as values.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which executes code inn a script.
